# ENGLISH
## How to Install the Theme (.jar) in IntelliJ IDEA

### Installation Steps:

1. **Download the file**  
   Download the `.jar` theme file.

2. **Install the plugin**  
   - Open IntelliJ IDEA.
   - Go to **File → Settings** (`Ctrl + Alt + S`).
   - In the left panel, select **Plugins**.
   - In the top right corner, click the ⚙️ (gear) icon.
   - Select **Install Plugin from Disk...**

3. **Select the file**  
   Browse and select the `.jar` file you downloaded, then click **OK**.

4. **Restart IntelliJ**  
   IntelliJ will prompt you to restart the IDE to apply the changes. Click **Restart IDE**.

5. **Activate the theme**  
   After restarting:
   - Go to **File → Settings → Appearance & Behavior → Appearance**.
   - Under **Theme**, select your new theme (usually named **Tokyo Dark Islands** or the name you set).
   - Click **Apply** and **OK**.

### Notes:
- Make sure the file has the `.jar` extension.
- This method works for both IntelliJ IDEA Community and Ultimate editions.
- If the theme doesn't appear after restarting, try restarting IntelliJ again or check that the `.jar` was installed correctly under **Plugins**.

Ready! You should now be able to enjoy your customized theme.

# SPANISH
## Cómo instalar el tema (.jar) en IntelliJ IDEA

### Pasos de instalación:

1. **Descarga el archivo**  
   Descarga el archivo `.jar` del tema.

2. **Instala el plugin**  
   - Abre IntelliJ IDEA.
   - Ve a **File → Settings** (`Ctrl + Alt + S`).
   - En el panel izquierdo, selecciona **Plugins**.
   - En la parte superior derecha, haz clic en el icono de ⚙️ (engranaje).
   - Selecciona **Install Plugin from Disk...**

3. **Selecciona el archivo**  
   Busca y selecciona el archivo `.jar` que descargaste y haz clic en **OK**.

4. **Reinicia IntelliJ**  
   IntelliJ te pedirá que reinicies el IDE para aplicar el tema. Haz clic en **Restart IDE**.

5. **Activa el tema**  
   Después de reiniciar:
   - Ve a **File → Settings → Appearance & Behavior → Appearance**.
   - En **Theme**, selecciona el nuevo tema (normalmente aparecerá como **Tokyo Dark Islands** o el nombre que hayas puesto).
   - Aplica los cambios.

### Notas:
- Asegúrate de que el archivo termine en `.jar`.
- Este método funciona en IntelliJ IDEA Community y Ultimate.
- Si el tema no aparece después de reiniciar, prueba reiniciar IntelliJ de nuevo o verifica que el `.jar` esté correctamente instalado en **Plugins**.

¡Listo! Ahora deberías poder disfrutar de tu tema modificado.
